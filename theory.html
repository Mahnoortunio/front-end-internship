Part A — Front-End Development Theory

Q1. What is Front-End Development?

Front-end development means creating the visible part of a website that users directly interact with.

A front-end developer designs and builds web pages using technologies like HTML, CSS, and JavaScript.

They make sure the website looks attractive, loads fast, and works properly on all devices.

Front-end focuses on user experience (UX) and user interface (UI).

It is important because it connects the website’s design with its functionality, allowing users to easily navigate and use the site.

Q2. What are HTML, CSS, and JavaScript?

Technology Explanation Example Use
HTML (HyperText Markup Language) HTML is used to create the basic structure or layout of a webpage. It defines elements like headings, paragraphs, links, and images. Add buttons, headings, and paragraphs
CSS (Cascading Style Sheets) CSS adds design and style to the webpage. It controls colors, fonts, spacing, and layout to make the website look attractive. Apply colors, fonts, spacing, and animations
JavaScript JavaScript adds interactivity and logic to the website. It helps create dynamic content like sliders, pop-ups, and form validation. Form validation, image sliders, and dynamic content
